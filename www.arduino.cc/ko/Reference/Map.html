map(value, fromLow, fromHigh, toLow, toHigh)

설명

한 범위에 있는 숫자 값을 다른 범위의 값으로 바꿉니다. 즉, fromLow 값은 toLow 값으로, fromHigh 값은 toHigh로, 중간 값들은 toLow와 toHigh의 중간 값들로 바뀌게 됩니다.

범위를 벗어나는 값들을 종종 의도적으로 사용할 수 도 있고 유용하기 때문에 값을 제한하지 않습니다. 만약 범위 제한이 필요하면 constrain() 함수를 사용할 수 있습니다.

범위의 "하한"을 "상한" 보다 크거나 작게 할 수 있기 때문에 map() 함수를 숫자의 범위를 반대로 뒤집는데 사용할 수 도 있습니다. 예로

y = map(x, 1, 50, 50, 1);

이 함수는 음수 또한 잘 처리하여, 다음 예

y = map(x, 1, 50, 50, -100);

또한 유효하고 잘 동작합니다.

map() 함수는 정수 연산을 사용하여 소수점 이하의 숫자를 만들어 내지 않습니다. 소수 나머지는 잘리며 반올림되거나 평균되지 않습니다.

매개 변수

value: 바꿀 숫자 값

fromLow: 값의 현재 범위의 하한 값

fromHigh: 값의 현재 범위의 상한 값

toLow: 바꾸고자 하는 범위의 하한 값

toHigh: 바꾸고자 하는 범위의 상한 값

결과 값

바뀐 숫자 값.

예

/* 아날로그 값을 8 비트 (0 ~ 255) 값으로 바꿉니다 */
void setup() {}

void loop()
{
  int val = analogRead(0);
  val = map(val, 0, 1023, 0, 255);
  analogWrite(9, val);
}

추가

수학적 재능을 갖춘 분을 위하여 여기 함수 전체가 있습니다. 

long map(long x, long in_min, long in_max, long out_min, long out_max)
{
  return (x - in_min) * (out_max - out_min) / (in_max - in_min) + out_min;
}
