로봇 라이브러리

로봇(Robot) 라이브러리는 아두이노 IDE 버전 1.0.5 이상에 포함되어 있습니다.

로봇에는 센서와 작동기(액추에이터, actuator)가 내장되어 있습니다. 라이브러리는 로봇 기능을 쉽게 사용할 수 있도록 설계되었습니다.

로봇은 모터 보드와 제어 보드 2개의 보드를 지니고 있으며, 각각의 보드들은 별도의 분리된 프로그램을 할 수 있는 프로세서를 가지고 있습니다.

이 라이브러리로 제어 보드에 있는 다양한 센서 그리고 주변 장치들과 인터페이스 할 수 있습니다:

  • 가변저항기(전위차계)
  • 5 순간 스위치들
  • 160x120 픽셀 컬러 화면
  • 512Kbit EEPROM
  • 스피커
  • 나침판
  • 3개의 I2C 커넥터
  • 8개의 TinkerKit 입력 커넥터

또한 모터 보드와 여러 작업을 할 수 있습니다:

  • 모터 속도와 방향 제어
  • 각각의 모터에서 사용되는 전류 감지
  • 5개의 바닥 감지 센서(선 감지센서라고도 함)
  • 보드의 I/O 핀들 사용
  • I2C 포트 제어
  • 4개의 TinkerKit 입력 상태 읽기

로봇에 대한 보다 자세한 내용은 시작 안내와 하드웨어 페이지들을 참조하세요.

라이브러리 구조

이 라이브러리는 아두이노 로봇의 기능에 쉽게 접근할 수 있게 해줍니다. Fat16, EasyTransfer, Squawk, 그리고 IRRemote를 포함한 다수의 타사 라이브러리들을 사용하며 또한 TFT, SPI, 그리고 Wire와 같은 여러 아두이노 라이브러리들을 사용합니다. 코드 크기를 최적화하기 위하여 로봇 기능은 로봇 라이브러리 안에 복제되어 있습니다.

제어 보드와 모터 보드 둘 모두 프로그램할 수 있지만, 초보 프로그래머는 제어 보드부터 시작하고 모터 보드는 나중으로 남겨 놓는 것이 좋습니다. 하나의 객체로 두 보드의 센서들을 사용할 수 있습니다.

로봇을 명령하는 두 주요 클래스들이 있습니다:

  • RobotControl: 기본 펌웨어를 실행할 때 제어 보드 뿐만 아니라 모터 보드의 I/O들과 모터들에도 명령합니다

  • RobotMotor: 모터 보드에 명령합니다. 이것을 이용하여 모터 보드 펌웨어의 사용자 정의 버전을 만드세요

예제들

기본 기능을 사용하여 빠르게 시작하고 동작시킬 수 있도록 로봇의 가능성들을 예시하여 줍니다.

  • Logo - 보드에 있는 키보드로 로봇이 어디로 갈지를 알려줍니다
  • Line Following - 경주 트랙을 그리고 트랙에서 로봇을 달리게 합니다
  • Disco Bot - 로봇을 8 비트 주크 박스로 바꾸고 비트에 맞춰 춤을 추게 합니다
  • Compass - 디지털 나침판으로 보물 찾기를 계획합니다
  • Inputs - 손잡이와 키보드를 어떻게 조절하는지 배웁니다
  • Wheel Calibration - 보다 나은 성능을 위하여 휠을 조정합니다
  • Runaway Robot - 거리 센서로 술래 잡기 놀이를 합니다
  • Remote control - 떨어저 있는 로봇에게 명령하기 위하여 옛 TV 리모콘을 다시 사용하세요
  • Picture browser - 자신의 이미지를 사용하고 싶은가요? 이것이 방법입니다
  • Rescue - 미로에 숨계진 진주 찾기 훈련을 로봇에게 시키세요
  • Hello User - 로봇 환영 데모를 고쳐 자신의 것으로 만듭니다
