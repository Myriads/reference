sizeof()

설명

sizeof 연산자는 변수 유형의 크기를 바이트 크기로 알려 주거나 배열이 차지하고 있는 바이트들의 숫자를 알려줍니다.

문법

sizeof(variable)

매개 변수

variable: 모든 데이터 유형 혹은 배열 (예 int, float, byte)

예제 코드

sizeof 연산자는 배열을 고쳐 배열의 크기가 변경되어도 배열을 변수로 사용하고 있는 프로그램의 다른 부분을 고치지 않고 사용할 수 있게 하여, 문자열과 같은 배열을 다루는데 유용합니다.

이 프로그램은 한번에 문자열에서 한 문자를 출력합니다. myStr 문구를 변경하여 보세요.

char myStr[] = "시험 문구입니다";
int i;

void setup(){
  Serial.begin(9600);
}

void loop() {
  for (i = 0; i < sizeof(myStr) - 1; i++){
    Serial.print(i, DEC);
    Serial.print(" = ");
    Serial.write(myStr[i]);
    Serial.println();
  }
  delay(5000); // 프로그램 속도를 늦춥니다
}
