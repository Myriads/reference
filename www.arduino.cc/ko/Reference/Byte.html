byte

설명

한 바이트(byte)는 부호를 사용하지 않는 8비트 값, 즉 0에서 255의 값을 저장합니다.

예

    byte b = B10010;  // "B"는 이진 수를 나타내기 위한 형식 지정자로 (B10010 = 18 십진수 값)
