{} 중괄호

중괄호(영어로 "curly braces" 간단하게 "braces" 혹은 "curly brackets"이라고도 함)는 C 프로그래밍 언어의 중요한 부분입니다. 중괄호는 밑에서 간략하게 보여준, 몇가지 다른 구조로 사용되며, 때로는 초보자에게 혼동을 줄 수도 있습니다.

'{' 여는 중괄호 다음에는 '}' 닫는 중괄호가 꼭 와야 합니다. 이것은 종종 중괄호가 짝이 맞춰진 상태(균형이 잡힌 상태)라고 합니다. 아두이노 IDE(Integrated Development Environment: 통합 개발 환경)에서는 중괄호의 짝이 맞았는지 확인할 수 편리한 기능이 있습니다. 중괄호를 선택하거나 중괄호 바로 뒤에 커서를 가져다 놓고 마우스를 클릭하면, 해당 논리적인 중괄호 짝에 강조 표시가 나타납니다.

현재 이 기능엔 IDE가 종종 "주석 처리된" 문장에서 중괄호를 (잘못) 찾는 것과 같은 약간의 버그가 있습니다.

초보 프로그래머들과 BASIC 언어에서 C로 온 프로그램머들은 중괄호 사용이 종종 혼란스럽고 어렵다는 것을 발견합니다. 결과적으로, C/C++에서의 {} 중괄호가 BASIC 언어의 서브 루틴(함수)에서 RETURN, 조건문에서 ENDIF, 그리고 FOR 반복문에서 NEXT를 대체합니다.

중괄호 사용이 매우 다양하기 때문에, 여는 중괄호를 입력하자마자 닫는 중괄호를 입력하는 것이 좋은 프로그래밍 습관입니다. 그리고 중괄호 사이에 몇개의 캐리지 리턴들을 넣고 실행문들을 넣는 것으로 시작하세요. 중괄호와 당신의 이러한 태도는 결코 짝이 맞기 않게 되는 경우가 발생하지 않을 것입니다.

짝이 맞지 않는 중괄호는 종종 아주 큰 프로그램에서 추적하기 어려운 이해할 수 없는 컴파일러 오류들을 발생시킬 수 있습니다. 다양한 용도로 인하여, 중괄호는 프로그램의 문법에 있어 매우 중요하며 중괄호를 한줄 혹은 두 줄 이동하는 것이 프로그램의 의미에 간혹 큰 영향을 미칩니다.

중괄호의 주요 사용법

함수

  void myfunction(데이터유형 인수){
    실행문
  }

반복문

  while (논리 표현식)
  {
     실행문
  }

  do
  {
     실행문
  } while (논리 표현식);

  for (초기화; 계속 실행 조건; 증가 혹은 감소 표현식)
  {
     실행문
  }

조건문

  if (논리 표현식)
  {
     실행문
  }

  else if (논리 표현식)
  {
     실행문
  } 
  else
  {
     실행문
  }
