비트 AND (&), 비트 OR (|), 비트 XOR (^)

비트 AND (&), 비트 OR (|), 비트 XOR (^)

이 비트 연산자들은 계산을 변수의 비트 수준에서 실행합니다. 비트 연산자들은 광범위한 범용 프로그램 문제들을 해결하는데 도움이 됩니다. 밑에 설명된 대부분이 비트 수학 자습서에서 가지고 온 것 들입니다.

설명과 문법

다음은 모든 연산자들에 대한 설명과 문법입니다. 보다 자세한 내용은 앞에서 설명한 자습서를 참고하세요.

비트 AND (&)

C/C++의 비트 AND 연산자는 두 다른 정수 표현식 사이에 하나의 앰퍼샌드(&)가 사용됩니다. 두 입력 비트가 모두 1이면 결과 출력 값이 1이고 그렇지 않은 경우 출력은 0인 규칙에 따라 비트 AND는 둘러싸인 식의 각 비트 위치에서 독립적으로 동작합니다. 이것을 표현하는 또 다른 방법: 

    0  0  1  1    피연산자1
    0  1  0  1    피연산자2
    ----------
    0  0  0  1    (피연산자1 & 피연산자2) - 반환되는 결과 값

아두이노에서, int 데이터 유형은 16 비트 값이므로 두 int 표현식 사이에 &를 사용하면 동시에 16개의 AND 연산이 이뤄집니다. 다음과 같은 코드 조각에서: 

    int a =  92;    // 2 진수(바이너리) 값: 0000000001011100
    int b = 101;    // 2 진수(바이너리) 값: 0000000001100101
    int c = a & b;  // 결과:    0000000001000100, 또는 십진수 68.

a와 b의 16 비트 각각의 비트들이 비트 AND를 사용하여 처리되며 16 결과 비트 모두가 c에 저장되어 2진 값이 01000100 이 되며 십진수로 68이 됩니다.

비트 AND의 가장 일반적인 용도 중 하나는 종종 비트 매스킹이라 불리우는 정수 값에서 특정 비트(또는 비트들)를 선택하는 것입니다. 예제는 아래를 참조하세요

비트 OR (|)

C/C++의 비트 OR 연산자는 수직 막대 기호인 | 입니다. & 비트 AND 연산자와 같이, | 비트 OR 연산자는 감싸고 있는 2개의 정수 표현식의 각각의 비트들을 독립적으로 조작하지만 다르게 동작합니다. 입력 비트 중 하나 또는 모두가 1이면 두 비트의 비트 OR는 1이고, 그렇지 않으면 0입니다. 즉: 

    0  0  1  1    피연산자1
    0  1  0  1    피연산자2
    ----------
    0  1  1  1    (피연산자1 | 피연산자2) - 반환된 결과

C++ 코드의 부분에 사용된 비트 OR 예입니다: 

    int a =  92;    // 2진수(바이너리) 값: 0000000001011100
    int b = 101;    // 2진수(바이너리) 값: 0000000001100101
    int c = a | b;  // 결과:    0000000001111101, 혹은 십진수 125.

아두이노 우노 예제 프로그램

비트 AND와 비트 OR 연산자의 일반적인 작업은 프로그래머가 포트에서 값을 읽고-수정하고-쓰는(Read-Modify-Write) 작업입니다. 마이크로컨트롤러에서, 1개의 포트는 8개의 핀들로 구성되며 핀들을 8 비트로 된 숫자로 제어할 수 있게 사용됩니다. 포트에 값을 씀으로써 동시에 8개 핀들을 제어할 수 있습니다.

PORTD는 디지털 핀 0,1,2,3,4,5,6,7의 출력 상태를 참조할 수 있는 내장 상수입니다. 비트 위치의 값이 1이면, 그 핀이 HIGH라는 의미입니다. (핀은 미리 pinMode() 명령을 사용하여 출력 모드로 설정하여야 합니다) 그래서 PORTD = B00110001;를 쓰면, 디지털 핀 0, 4 그리고 5를 HIGH로 만들 수 있습니다. 여기에 시리얼 통신을 위하여 아두이노에서 사용되는 핀 0과 1의 상태를 또한 바꿀 수 있는 작은 문제가 있어 시리얼 통신을 방해할 수 있습니다. 

     프로그램에서 사용된 알고리즘은 다음과 같습니다:
  • PORTD 값을 읽어 들이고 비트 AND 연산자로 제어하고자 하는 핀에 해당되는 비트들만 지웁니다.
  • 비트 OR 연산자로 제어하고자 하는 새로운 값과 수정된 PORTD 값을 결합합니다. 

int i;     // 카운터 변수
int j;

void setup(){
DDRD = DDRD | B11111100; // 핀 0와 1은 그대로 나두고 핀 2 ~ 7를 출력 모드로 설정합니다(xx | 00 == xx)
// 핀 2에서 7에 pinMode(pin, OUTPUT) 명령을 사용한 것과 같습니다
Serial.begin(9600);
}

void loop(){
for (i=0; i<64; i++){

PORTD = PORTD & B00000011;  // 핀 0와 1은 그대로 나두고 2 ~ 7 비트 값들을 지웁니다 (xx & 11 == xx)
j = (i << 2);               // 핀 0와 1은 피하기 위하여 변수 값을 핀 2 ~ 7으로 이동 시킵니다.
PORTD = PORTD | j;          // LED 핀의 새로운 정보와 포트 정보를 결합니다
Serial.println(PORTD, BIN); // 디버그를 위하여 PORTD 값을 출력합니다
delay(100);
   }
}

비트 XOR (^)

C/C++에 배타적 비트 OR라 불리우는 다소 특이한 비트 XOR 연산자가 있습니다. (영어로 일반적으로 "엑스-오어"로 발음됩니다.) 비트 XOR 연산자는 꺽쇠(웃꺽쐬, caret) 기호 ^를 사용하여 작성됩니다. 이 연산자는 | 비트 OR 연산자와 매우 유사합니다, 오직 두 입력의 주어진 비트 위치의 값이 같이 모두 0이거나 1인 경우에만 0이 됩니다: 

    0  0  1  1    피연산자1
    0  1  0  1    피연산자2
    ----------
    0  1  1  0    (피연산자1 ^ 피연산자2) - 반환된 결과

비트 XOR를 보는 다른 방법으로 두 입력 비트가 서로 다른 경우 결과는 1이되고, 서로 같으면 0가 된다는 것입니다.

여기 간단한 코드 예제입니다: 

    int x = 12;     // 2진: 1100
    int y = 10;     // 2진: 1010
    int z = x ^ y;  // 2진: 0110, 혹은 십진수 6

^ 연산자는 종종 정수의 일부 비트들 값을 바꾸기(예로 0은 1로, 혹은 1은 0로 바꾸는) 위하여 사용됩니다. 비트 XOR 연산에서 만약 마스크 비트가 1이면, 그 비트 값은 반전되며; 비트 마스크가 0이면, 그 비트 값은 반전되지 않고 같은 값을 지니게 됩니다. 다음은 디지털 5번 핀을 깜빡이는 프로그램입니다. 

// Blink_Pin_5
// 배타적 OR (XOR) 데모
void setup(){
DDRD = DDRD | B00100000; // 디지털 5번 핀을 출력 모드로 설정합니다
Serial.begin(9600);
}

void loop(){
PORTD = PORTD ^ B00100000;  // 비트 5(디지털 5번 핀)을 반전시키고 다른 핀들은 그대로 나둡니다
delay(100);
}
